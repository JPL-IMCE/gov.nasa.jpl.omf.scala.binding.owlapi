abstract class OWLAPIOMFGraphStoreMetadata extends AnyRef

This is a legacy OWL-based metadata for OMF models. It is refactored out of the OWLAPIOMFGraphStore. It will be eliminated when this legacy metadata is no longer needed for the ontology processing workflow.
